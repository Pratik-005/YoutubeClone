
# YoutubeClone

<p align="justify">
It is a full-stack web application developed using the MERN (MongoDB, Express.js, React, Node.js) stack. Inspired by the popular video-sharing platform, this project demonstrates my proficiency in building scalable and feature-rich applications. The YouTube Clone provides users with a familiar and immersive experience of uploading, sharing, and watching videos, making it an ideal addition to my portfolio.
</p>


## Features

- Interactive User Interface
- Video Discovery
- Video Playback

## Installation

1. Clone the repository:

```
git clone https://github.com/Pratik-005/YoutubeClone.git
```


2. Get a free API Key at https://rapidapi.com/hub

3. Install dependencies:

```
npm install
```
4. Enter your API key in .env file
```
const REACT_APP_RAPID_API_KEY = 'ENTER YOUR API';
```

5. Start application using following command:
```
npm run start
```
